# ActivityTrackerApp
Cross platform mobile application made with Xamarin.Forms for managing sport activities.

# Stack
- Xamarin.Forms
- Xamarin Essentials
- Autofac
- LottieXamarin
- Rg.Plugins.Popup

# Features
It offers:
- Managing user accounts
- Managing sport activities
- Synchronising activities with external provider - Strava
